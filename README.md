# Inserts-Paper-Generator
このツールは、複数の画像を読み込み、指定したサイズ（高さ・横幅）で用紙内に自動整列させてPDFとして書き出すツールです。

🚀 基本的な使い方の流れ

画像の追加

サイドパネルの「写真を追加」エリアをクリックしてファイルを選択するか、画像を直接ドラッグ＆ドロップしてください。

読み込んだ画像が横長（幅 > 高さ）の場合、自動的に90度回転して縦長として配置されます。

レイアウト設定の調整

用紙サイズ: A4, B5, ハガキなどから選択します。

画像の高さ: 配置する画像の縦の長さを指定します（初期値: 9cm）。

アスペクト比を無視して横幅を指定:

チェックを入れると、画像の比率を無視して強制的に指定した横幅で配置します。

チェックを外すと、高さに合わせて元の比率を維持したまま配置されます。

余白・間隔: 用紙の端からの距離や、画像同士のスキマを調整します。

プレビューの確認

右側のプレビューエリアに、実際の印刷イメージが表示されます。

画面サイズに合わせて自動縮小されますが、印刷時は設定した実寸（mm/cm）で書き出されます。

PDFの作成

「PDFを作成」ボタンを押すと、現在のレイアウトがPDFとしてダウンロードされます。

🛠 便利な機能

1. 入力欄のスマート機能

数字入力ボックス（高さ、横幅など）は、入力中に内容をすべて消去（空白に）することができます。

空白の間はデフォルト値が仮適用され、入力欄からフォーカスを外すと自動的にデフォルト値が入力されます。

2. 横幅のプリセット（ペン回し向け）

「アスペクト比を無視」を有効にすると、以下の専用ボタンが表示されます。

G3ボタン: 横幅を 2.5cm に設定します。

rsvpボタン: 横幅を 2.2cm に設定します。

3. 個別画像の操作

プレビュー上の画像、または画像一覧のアイテムにあるアイコンで以下の操作が可能です。

🔄 回転アイコン: 画像をさらに90度回転させます。

🗑 ゴミ箱アイコン: その画像だけを削除します。

4. 設定のリセットと全消去

リセットボタン: レイアウト設定（サイズや余白）をすべて初期状態に戻します。

全消去ボタン: 読み込んだすべての画像を一度に削除します。

💡 ヒント

単位の切り替え: 「cm」と「mm」を切り替えて、使いやすい単位で数値を入力できます。

枠線の表示: 「枠線を表示」にチェックを入れると、切り取りの目安となる点線が表示されます。

印刷時の注意: PDFを印刷する際は、プリンターの設定で**「実際のサイズ（拡大縮小なし）」**を選択して印刷してください。
